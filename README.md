# Web Calculator

A simple and responsive **web-based calculator** built using **HTML, CSS, and JavaScript**.  
This calculator performs basic arithmetic operations and supports keyboard input for faster usage.

---

## Features

- Perform basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
  - Modulus (%)
- Clean and responsive user interface
- Keyboard input support
- Clear (AC) and Delete functionality
- Prevents multiple decimal points
- Handles division by zero errors

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How It Works

The calculator keeps track of:

- **currentInput** – the number currently being entered
- **previousInput** – the previous number stored when an operator is selected
- **operator** – the selected arithmetic operation

When the user enters numbers and selects an operator, the calculator performs the calculation and displays the result.

---

## Keyboard Shortcuts

| Key | Action |
|----|------|
| 0 – 9 | Enter numbers |
| . | Decimal |
| + - * / | Operators |
| Enter / = | Calculate result |
| Backspace | Delete last digit |
| Escape | Clear all |
